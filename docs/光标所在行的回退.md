
## displayBuffer() 每次被调用时 
{
     bloc.Y = w.StartLine.Line         // buffer 行号 ← 你说的 line 对的就是这个
     vloc.Y = 0                        // screen 行号（窗口里的相对位置）
     if softwrap: vloc.Y = -w.StartLine.Row  // softwrap 时 vloc.Y 可能从负数开始

     准备 60+ 行局部变量（matchingBraces、tabsize、showchars 解析、colorcolumn…）
     定义 5 层闭包：getRuneStyle / draw / wrap / …  ← ★ 遗漏点 2（闭包依赖上面那些变量）

     while (vloc.Y < w.bufHeight && bloc.Y < b.LinesNum()) {  // ★ 双重终止条件
         1. 决定这行是不是光标行（影响行号样式）
         2. 画 gutter（行号 / diff gutter / message gutter），仅当 vloc.Y >= 0
         3. 读 b.LineBytes(bloc.Y)，解 rune 循环
            - 解一个 rune → 算 width → 累计 totalwidth
            - 调 getRuneStyle 决定这个 rune 最终显示成什么字符 + 什么样式
            - 调 draw 把它写到 screen.SetContent
            - ★ 遗漏点 3：如果 softwrap + 词太长，draw 内部会把 vloc.Y++ 并 wrap()，
              一行 buffer 可能消费多个 vloc.Y（这时 bloc.Y 不变）
         4. EOL 填充：用空格填到 maxWidth（带 cursorline/colorcolumn 样式）
         5. bloc.Y++   ← ★ 实际是循环末尾 bloc.Y 才自增，不是循环开头
            vloc.Y++（外层 for 头里）
     }
}


## displayBufferMD() 应当是这样的

例：
- 文件总共有100行，共有5个segments，每个都是20行
- 当前screen 是从buffer.line15-line30
- 则 displayBufferMD()里面的segments = [s0裁[15,19],  s1裁[20,30]]


displayBufferMD() 每次被调用时 {
	一些准备工作
    segments = filterSegmentsToVisible(b.MDSegments, visibleStart, visibleEnd)
    //  这里 [s0裁[15,19],  s1裁[20,30]]

    vY = 0
    for seg in segments {
    	// 说明：多光标时，任何一个光标落在seg里面就算是 cursor inside this seg
    	if cursor inside this seg {
			micro原生处理()    		
    	}else {
    		microNeo处理()
    	}
        w.mdCache.append({BufStartLine, BufEndLine, RowCounts})
    }

    // 填剩余空间
    for ; vY < bufHeight; vY++ {
        w.drawGutterAndLineNumMD(vY, -1)
        for col := 0; col < bufWidth; col++ screen.SetContent(... ' ')
    }
}

microNeo处理() {
        lines    = w.linesFromBuffer(seg.BufStartLine, seg.BufEndLine)
        rendered = seg.Render(lines, bufWidth, w.mdConfig)   // MD 渲染器排好版

        // 关键修正：相对 BufLine → 绝对 BufLine
        for row in rendered.Rows {
            row.BufLine += seg.BufStartLine
            for cell in row.Cells { if cell.BufLine >= 0: cell.BufLine += seg.BufStartLine }
        }

        for row in rendered.Rows {
            if (vY >= bufHeight) break
            w.drawGutterAndLineNumMD(vY, row.BufLine)        // bufLine=-1 走空行分支

            lastFg, lastAttr, hasLast = 0, 0, false         // LineMatch 稀疏，沿用上次
            for col, cell in row.Cells {
                if (screenX 越界) continue
                style = cell.Style
                if (cell.BufLine >= 0 && cell.BufX >= 0) {
                    if (group := w.Buf.Match(cell.BufLine)[cell.BufX]; ok) {
                        lastFg, _, lastAttr = config.GetColor(group).Decompose()
                        hasLast = true
                    }
                    if (hasLast) style = 合并(lastFg, lastAttr, 保留 bg)
                }
                screen.SetContent(...)
            }
            vY++                                              // ★ bloc.Y 不自增
        }
	
}
