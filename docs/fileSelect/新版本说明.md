# Release 1.1.13 更新说明

用microNeo代替Linux的cd命令
- Linux的cd命令在多层目录的时候实在是太难用了
- 所以这版的microNeo学习了yazi的方法，可以在fileManager里面切换目录。退出microNeo的时候自动把当前目录切换到最后的文件的所在目录里

## 具体使用方法如下：

在zshrc/bashrc里面添加下面的代码：
```zsh
function m() {
    local tmp="$(mktemp -t "microneo-cwd.XXXXXX")" cwd
    command microneo "$@" --cwd-file="$tmp"
    IFS= read -r -d '' cwd < "$tmp"
    [ "$cwd" != "$PWD" ] && [ -d "$cwd" ] && builtin cd -- "$cwd"
    rm -f -- "$tmp"
}
```

然后使用 m 命令打开microNeo，选择编辑某个文件。退出后系统目录就自动切换为文件的所在目录了

# Release 1.1.13 Release Notes (English)

Use microNeo in place of Linux's `cd` command.
- Navigating deep directory trees with plain `cd` is painful.
- This release borrows yazi's approach: you can switch directories inside the file manager, and when you quit microNeo the shell automatically lands in the directory of the last file you had open.

## How to use

Add the following to your `~/.zshrc` / `~/.bashrc`:

```zsh
function m() {
    local tmp="$(mktemp -t "microneo-cwd.XXXXXX")" cwd
    command microneo "$@" --cwd-file="$tmp"
    IFS= read -r -d '' cwd < "$tmp"
    [ "$cwd" != "$PWD" ] && [ -d "$cwd" ] && builtin cd -- "$cwd"
    rm -f -- "$tmp"
}
```

Then open microNeo with `m`, pick a file to edit, and on quit your shell's working directory switches to that file's directory.
