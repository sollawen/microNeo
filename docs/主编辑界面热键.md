# microNeo 主编辑界面热键清单

---

## 阅读须知

- microNeo 几乎不重映射 micro 的原生热键。**例外**：`Ctrl-o` 从原生的 `OpenFile`（弹 InfoBar 输入框）改为直接触发 `:file`（弹 FileSelector）；其余 micro 原生热键全部保留。用户可在 `~/.config/microNeo/bindings.json` 自行改回 `OpenFile`。
- microNeo **唯一自加**的特殊行为是 **ESC 切到阅读模式**——按 ESC 时光标所在 segment 保持 MD 渲染，按其他任何键又回到编辑模式（光标段回退 micro 原生）。见 [§10 模式切换](#10-阅读--编辑模式切换-microneo-自加)。
- 部分键在 macOS 和其他平台默认绑定不同，文档**两套并列**。
- 用户可在 `~/.config/microNeo/bindings.json` 中改键，本文档列的是 **defaults**。
- 表格中的 `*` 表示"在 MD 文件下行为/视觉有差异"（详见 [§11](#11-md-文件下的行为差异)）。

## 4. 查找与替换

| 热键 | 动作 | 说明 |
|------|------|------|
| `Ctrl-f` | `Find` | 打开搜索栏（支持 regex）|
| `Alt-F` | `FindLiteral` | 字面量搜索（无 regex）|
| `Ctrl-n` | `FindNext` | 下一个匹配 |
| `Ctrl-p` | `FindPrevious` | 上一个匹配 |
| `Esc` | `UnhighlightSearch` | 清除搜索高亮（包含在 `Esc` 复合动作里）|

搜索栏打开后用 `Enter` 确认；`Ctrl-n/p` 切匹配；`Esc` 关闭。

**补充说明**：

- **`Ctrl-n` / `Ctrl-p` 在没搜过时无效**。源码 `actions.go:1203-1206`：`Buf.LastSearch == ""` 时 `FindNext` / `FindPrevious` 内部 early return，按下等于吞键——**光标不动、无提示、无错误**。所以 `Ctrl-n/p` 只能延续本 buffer 上一次的搜索。
- **`LastSearch` 是 buffer 级别**（`buffer.go:277`，注释"stores the last successful search"）。在 `a.md` 搜过 `hello` 后切到 `b.md`，`b.md` 的 `LastSearch` 是默认空串，`Ctrl-n` 在新 buffer 里同样哑火。想跨 buffer 复用搜索词，必须在新 buffer 里重新按 `Ctrl-f`（或 `Alt-F`）搜一次。
- **`Esc` 关闭搜索栏**会同时清掉搜索高亮（`UnhighlightSearch` 包含在 `Esc` 复合动作里），但**不会清空 `LastSearch`**——关闭搜索栏后 `Ctrl-n/p` 仍能继续翻匹配。想彻底清掉本次搜索记忆，只能切换 buffer。

> 这个Ctrl-f非常反人类，大部份人都只需要Alt-f

## 11. 阅读 / 编辑模式切换（microNeo 自加）

这是 microNeo **唯一**对热键行为有"非中立"影响的部分。

实现：`internal/action/bufpane_md.go` 的 `observeEditModeToggle`。  
触发点：`bufpane.go:532`，micro 原生处理完事件之后。

| 触发事件 | 模式切换 | 渲染结果 |
|----------|----------|----------|
| 打开 `.md` 文件 | → **编辑模式**（`editMode=true`）| 光标所在 segment 显示原字符（`#`、`\|`、原始代码块字符…）|
| 按 `Esc` | → **阅读模式**（`editMode=false`）| 全部 segment 走 MD 渲染（标题加粗、表格加边框、代码块着色…）|
| 按任意非 ESC 键 | → **编辑模式** | 光标所在 segment 回退原生，其他段保持 MD 渲染 |
| 鼠标 click / drag | → **编辑模式** | 落点 / 选区所在 segment 回退原生 |
| 鼠标滚轮 | **保持** | 滚动但不切模式 |
| 鼠标释放（无按钮）| **保持** | — |
| 单独 modifier 键 | 不发事件 → 不切 | — |

### 11.1 模式对视觉的影响

- **阅读模式**：所见即所得的渲染效果，**光标不显式高亮当前段**。
- **编辑模式**：光标所在段"剥掉"渲染，露出原始字符；其他段保持渲染。

### 11.2 多光标 / 选区的回退判定

`hasCursorInside`（`bufwindow_md.go:69`）同时检查：
1. **光标本身**所在行是否在 segment 内。
2. **选区**是否与 segment 有交集。

任一命中即回退原生。

### 11.3 切换的可控性

`observeEditModeToggle` 是 hard-coded，**没有暴露成 setting**。  
如果你想"永远保持阅读模式"或"永远保持编辑模式"，目前需要改源码。

## 12. MD 文件下的行为差异

虽然 micro 原生热键在 microNeo 中**没有重映射**，但视觉/行为会有这些差异：

| 热键 / 行为 | 在普通 .txt 文件 | 在 .md 文件 |
|------------|------------------|-------------|
| `↑` `↓` | 按 buffer 行移动 | 按 buffer 行移动，**跳过装饰行**（表格分隔、HR、代码块边框）|
| `↓` 在表格第 1 行 | 进表格第 2 行 | 进表格第 2 行的**内容行**（跳过 `\|---\|` 装饰行）|
| 鼠标点击装饰行 | — | 忽略，无效果 |
| 拖选跨段 | 全部原生 | 所选 segment 全部临时回退原生 |
| `Esc` | 清选区 | 清选区 **+ 切阅读模式** |
| 编辑后（输入 / 删除）| buffer 变 | buffer 变 + `DetectSegments` 重算（`buffer.go:1031-1032`）|
| `Ctrl-s` 保存 | 保存 | 保存，下次打开重算 segments |
| `Ctrl-z` 撤销 | 撤销 | 撤销，**触发 DetectSegments** |
| 多光标编辑 | 全部原生 | 多光标命中时各 cursor 段全部回退原生 |

> "跳过装饰行"是因为 `BufLine = -1` 的 cell 不在 buffer 行号空间内，光标 / 选区永远不会包含它们。

## 13. 补充说明

### 13.1 关于 "动作链"（multi-action）

`defaults_*.go` 里的字符串形如 `"Escape,Deselect,ClearInfo,RemoveAllMultiCursors,UnhighlightSearch"`。  
这是 **multi-action**：按 `Esc` 一次会**依次执行**所有列出的 action。  
类似的还有 `Tab` = `Autocomplete|IndentSelection|InsertTab`（按顺序尝试，第一个不适用就执行下一个）。

### 13.2 关于 "type-to-find"

micro 的搜索栏有 type-to-find 行为：搜索栏打开后输入字符就开始搜索。  
这跟"主编辑界面直接输入字符"是两套独立事件，**与 microNeo 的模式切换不冲突**。

### 13.3 改键建议

`~/.config/microNeo/bindings.json` 格式：

```json
{
  "Ctrl-s":          "Save",
  "Ctrl-Shift-s":    "SaveAs",
  "Ctrl-MouseWheelUp":   "ScrollUp",
  "Alt-n":           "SpawnMultiCursor",
  "F1":              "ToggleHelp"
}
```

改键不会触发 microNeo 的"重编译 segments"——热键系统和 MD 渲染管线是解耦的。

### 13.4 microNeo 默认配置相关

打开 `~/.config/microNeo/settings.json` 看到的 `md*` 开头选项（如 `mdtablealign`、`mdbolditalic`）**与热键无关**，只影响 MD 渲染外观。

## 15. 速查清单

microNeo 自身**没有新增任何热键**（详见 §11），以下清单是 micro 在 `internal/action/defaults_*.go` 的全部默认键位。格式：

- `热键` — 一句话功能
- 标注 `*` = 在 MD 文件下有特殊行为（见 §12）
- 标注 `❌` = 文档作者认为可以去掉的热键（对应正文 §3-§9 中的意见）

### 15.1 光标移动（基础）

- `←` `→` `↑` `↓` — 单字符 / 单行移动
- `Home` — 跳到行首（toggle：非空白 → 空白前 → 绝对行首）
- `End` — 跳到行尾
- `Ctrl-Home` — 跳到文件首
- `Ctrl-End` — 跳到文件尾
- `PageUp` `PageDown` — 上下翻页
- `Ctrl-l` — 弹输入框跳到指定行（`:goto N`）

### 15.2 光标移动（词级 / 段级）

- `Ctrl-←` `Ctrl-→` — 按词左右移动
- `Alt-{` `Alt-}` — 上一段 / 下一段（空行分隔）
- `Alt-↑` `Alt-↓` — **整行** 上移 / 下移

### 15.3 光标移动（Emacs 风格）

- `Alt-f` — 按词右移
- `Alt-b` — 按词左移
- `Alt-a` — 到行首
- `Alt-e` — 到行尾

### 15.4 选择（基础）

- `Shift-←` `Shift-→` `Shift-↑` `Shift-↓` — 按字符 / 行选区
- `Shift-Home` — 选到行首（toggle）
- `Shift-End` — 选到行尾
- `Ctrl-Shift-Home` — 选到文件首
- `Ctrl-Shift-End` — 选到文件尾
- `Shift-PageUp` `Shift-PageDown` — 选区翻页
- `Ctrl-a` — 全选

### 15.5 选择（词级 / 文件级）

- `Ctrl-Shift-←` `Ctrl-Shift-→` — 选到上 / 下一词
- `Alt-Shift-←` `Alt-Shift-→` — 选到行首 / 行尾
- `Ctrl-Shift-↑` `Ctrl-Shift-↓` — 选到文件首 / 文件尾

### 15.6 文本编辑（插入 / 删除）

- `Enter` — 插入换行
- `Tab` — 自动补全 / 缩进选区 / 插入 Tab
- `Backtab` (`其他`) — 反向自动补全 / 反向缩进（即 `Shift-Tab`）
- `Backspace` `Ctrl-h` — 删除前一个字符（`Ctrl-h` ❌）
- `Delete` — 删除后一个字符
- `Alt-Backspace` `Alt-Ctrl-h` — 删除前一个词（`Alt-Ctrl-h` ❌）

### 15.7 行级操作

- `Alt-↑` `Alt-↓` (`其他`) — 整行上移 / 下移
- `Ctrl-d` — 复制行 / 选区
- `Ctrl-k` — 剪切行
- `Ctrl-t` — 新建 Tab

### 15.8 撤销 / 重做 / 复制 / 剪切 / 粘贴

- `Ctrl-z` — 撤销
- `Ctrl-y` — 重做
- `Ctrl-c` — 复制（无选区时复制当前行）
- `Ctrl-x` — 剪切（无选区时剪切当前行）
- `Ctrl-v` — 粘贴

### 15.9 查找与替换

- `Ctrl-f` — 打开搜索栏（支持 regex）❌
- `Alt-F` — 打开搜索栏（字面量，不支持 regex）
- `Ctrl-n` — 下一个匹配（本 buffer 内有效）
- `Ctrl-p` — 上一个匹配（本 buffer 内有效）

### 15.10 文件操作

- `Ctrl-s` — 保存
- `Ctrl-o` — 打开文件选择器（直接弹 FileSelector，等价于 `:file`）
- `Ctrl-q` — 退出

### 15.11 Tab / Split 导航

- `Alt-,` `Alt-.` — 上 / 下 Tab（边界时跳到首 / 末 Tab）
- `Ctrl-PageUp` `Ctrl-PageDown` — 上 / 下 Tab（同上）
- `Ctrl-w` — 切换到下一分屏（边界时跳回首分屏）

### 15.12 视图与界面

- `Ctrl-g` — 切换帮助面板 ❌（与 `Alt-g` 合并即可，二选一）
- `Alt-g` — 切换键位菜单 ❌（与 `Ctrl-g` 合并即可，二选一）
- `Ctrl-r` — 切换尺规 ❌（行号应常驻显示）
- `Ctrl-b` — Shell 模式：让出整个屏幕给一条命令（不写回 buffer）❌
- `Ctrl-e` — 命令模式（输入 microNeo / micro 命令）
- `Ctrl-u` — 切换宏录制
- `Ctrl-j` — 播放宏
- `Insert` — 切换改写模式 ❌
- `Alt-Enter` — 打开便笺面板

### 15.13 Diff 导航

- `Alt-[` — 跳到上一段 diff（`hunk`）
- `Alt-]` — 跳到下一段 diff

### 15.14 多光标

- `Alt-n` — 按内容扩展多光标（**第一次按只选词**，之后每按一次加一个光标 + 选区）
- `Alt-m` — 把每行 `X=0`（行首）加光标（缩进会丢）
- `Alt-Shift-↑` `Alt-Shift-↓` — 上 / 下一行视觉同列加光标（保留 tab / 中文 / 软换行的列对齐）*（推荐）
- `Alt-p` — 删最后一个多光标 ❌
- `Alt-c` — 清空所有多光标（与 `Esc` 等价）❌
- `Alt-x` — 跳到下一多光标 ❌
- `Ctrl-MouseLeft` — 鼠标点击位置加多光标

### 15.15 鼠标

- `MouseLeft` — 单击设置光标
- `MouseLeft Drag` — 拖拽产生选区
- `MouseLeft Release` — 释放鼠标（光标定位）
- `MouseMiddle` — 粘贴（X11 鼠标中键）
- `MouseWheelUp` `MouseWheelDown` — 垂直滚动
- `MouseWheelLeft` `MouseWheelRight` — 水平滚动

### 15.16 F 键（历史遗留，几乎用不到）

- `F2` — 同 `Ctrl-s`（保存）❌
- `F3` `F7` — 同 `Ctrl-f`（搜索）❌
- `F4` `F10` — 同 `Ctrl-q`（退出）❌

建议：在 `bindings.json` 里把 F2 / F3 / F4 / F7 / F10 都映射成 `None`，避免误触。

### 15.17 Esc 复合动作

- `Esc` — 一次性执行：退出当前模式 + 清选区 + 清信息条 + 清多光标 + 清搜索高亮

### 15.18 microNeo 自身

microNeo **没有新增任何键位绑定**，但有以下**事件旁路观察行为**（详见 §11）：

- `Esc` 按下 → 切到阅读模式（`editMode = false`）
- **任何非 Esc 按键**（含字母 / 方向键 / Enter / Ctrl / Alt 组合）→ 切到编辑模式（`editMode = true`）
- **鼠标点击 / 拖拽**（`Buttons() != ButtonNone` 且不含滚轮）→ 切到编辑模式
- **鼠标滚轮**（WheelUp / Down / Left / Right）→ **保持当前模式**
- **鼠标释放**（ButtonNone）→ **保持当前模式**
- **单独 modifier 键**（Shift / Ctrl / Alt / Meta）→ 终端通常不发事件，**不触发**
- 打开 `.md` / `.markdown` 文件 → **默认进编辑模式**（`bufpane_md.go:35`）
