# 下面我们完善一下位置计算

## 正常时候

notePane应当出现在当前光标的下面的row里
- 比如当前光标在row5，注意不是Line5，是row5，则notePane的边框从row6开始，内容从row7开始
- 如果当前有多光标，则从最下面的那个光标的下一个row开始
- 如果当前有select，则从select区域的最下面的下一个row开始
