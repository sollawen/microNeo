# Git Commit

## Rules
- Every commit MUST have a Conventional Commits prefix: `feat:`, `fix:`, `docs:`, `style:`, `refactor:`, `perf:`, `test:`, `chore:`, `ci:`, `revert:`, etc.
- Title in **English**, body is up to LLM
